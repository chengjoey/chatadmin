# ChatAdmin

用Golang和reids开发的客户聊天管理系统


## Installation
go get github.com/chengjoey/chatadmin

install redis and mofify chatadmin.yaml

```bash
./chatadmin
```

## Usage
admin和administrator为管理员用户, 可查看有哪些用户在线且能与之聊天
open browser http://127.0.0.1:3389/
